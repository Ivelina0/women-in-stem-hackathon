# Repository Instructions

This repository is a **static GitHub Pages site**.

## Tech constraints
- Use only **HTML, CSS, and JavaScript**.
- Keep solutions simple and beginner-friendly.

## Design direction
- Maintain a **playful, cartoonish, scrapbook, pastel** vibe.
- **Never** make the site look corporate, sterile, or startup-like.
- Reuse and preserve the project’s established color palettes.

## Content and editing approach
- Keep pages and code **beginner-editable** (clear structure, minimal complexity).
- Reuse content from existing Markdown files where possible instead of rewriting from scratch.
